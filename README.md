# Parking-Lot-Management_System
A Parking Lot Management System designed to manage vehicle entry, exit, and real-time slot allocation efficiently. The system applies object-oriented principles to ensure scalability and maintainability.

**Features**
1.Vehicle Entry & Exit: Track vehicles entering and leaving the parking lot.
2.Real-Time Slot Allocation: Automatically assigns available parking slots to incoming vehicles.
3.Slot Availability Tracking: Monitor the current status of each parking slot (occupied or free).
4.Object-Oriented Design: Uses abstraction, inheritance, and encapsulation.

**Technologies Used**
1.Programming Language:  C++
2.Design Principles: Object-Oriented Programming (OOP)
3.Data Structures: Arrays, Lists, Maps.
